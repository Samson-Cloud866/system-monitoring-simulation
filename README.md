# System Monitoring Simulation

This is a beginner DevOps project inspired by my background in healthcare and quality control.

The goal is to simulate how systems are monitored by checking uptime, CPU usage, memory usage, logging issues, and displaying alerts when thresholds are exceeded.

## Features
- Check system uptime
- Monitor CPU and memory
- Write logs to a file
- Output alert messages

## Why I built this
My background taught me the importance of monitoring processes, detecting issues early, and maintaining reliability. This project connects that mindset to DevOps.
